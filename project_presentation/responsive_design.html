About the assignment

This assignment is about applying CSS Flex and Grid methods to a mockup landing page design to make it a responsive layout. The student should download the necessary files and use CSS Flex, Grid, min width/height, and media queries to layout the page. Additionally, the header background and all divs with a class called something with “_graphics” should have an image as a background-image. The assignment is considered finished when it looks good on both desktop and mobile. The student is encouraged to add fonts, a font hierarchy, and colors to the design.

What have I learned?

In this assignment I didn’t learn much that i hadn’t heard before, but it was nice to get refreshed some of the topics, as well as putting everything together in one project.
